Hanoi Tower Visualisation

What is Tower of Hanoi?

  • It's a mathematical puzzle
  • Invented by french mathematician Edouard Lucas in 1883
  • It's a stack of discks of different sizes

What this puzzle is about?

  • We have three poles and some disks
  • All disks are initially arranged in ascending order in a pole
  • Problem is to move all disks to another pole

What rules should be followed?

  • Only one disk can be moved at a time
  • Bigger disk cannot be placed on smaller one
  • We need to find minimum number of moves to do so

Here I have made a programme that solves this puzzle and animate every move

Follow these steps

Give the input as number of disks

Also set the animation speed at which you want to run this animation

Now press the Start button

Press Reset before you give another input
